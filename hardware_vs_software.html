Hardware vs Software

The physical tools vs the digital magic.

What is Hardware?

Hardware refers to the physical parts of a computer that you can see and touch.

  • Keyboard & Mouse
  • Monitor & CPU
  • Motherboard & RAM

What is Software?

Software is a set of instructions that tells the hardware what to do. You cannot touch it!

  • Windows / macOS
  • Google Chrome
  • Minecraft / Roblox
